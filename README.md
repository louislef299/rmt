# rmt

A tilde file removal CLI for emacs users.

```go
go install github.com/louislef299/rmt@latest
```

CLI Summary:

```bash
$ rmt -h
Removes all files containing a tilde found

Usage:
  rmt [flags]

Flags:
  -h, --help          help for rmt
  -i, --interactive   interactive output
  -r, --recursive     walk filepath starting at current directory
```
